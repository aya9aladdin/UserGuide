Guessing¶

When a Universe is created from a Universe, MDAnalysis can guesses properties that have not been read from the file. Sometimes these properties are available in the file, but are simply not read by MDAnalysis. For example, masses are always guessed. The guesser module contains different context-specific guessers. This can be forcefield-specific like MartiniGuesser, or format-specific guesser like PDBGuesser. You can utilize guessers either by initiating an object of it or through the guess_TopologyAttributes() API of the universe to guess various properties to the universe. See Guessing topology attributes for details.
